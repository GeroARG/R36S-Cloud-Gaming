# R36S Cloud Gaming Project
### The aim of this project is to be able to play games like Wolfenstein 2: The New Colossus or GTA V right onto your handheld, in this case, R36S (I will probably port it to other consoles)

Frequent Questions:
How can I start playing these games onto my R36S?:
Well, to begin with, This project will be using Tailscale and Moonlight Game Streaming along with Sunshine to be able to connect to the virtual machine, here are the steps:
---

## Step 1

For this step, you will have to put these thingy onto your Google Cloud, to do so, create a new notebook, this is what you would see, click on "New Notebook".

## Step 2

Dwonload the following .ipynb file and then Copy and Paste its content onto your Google Colab run thing

https://drive.google.com/file/d/1TO3Is-qrXugqUVFbtxN86XQ_eFqNdIBq/view?usp=sharing

### Step 3

Ok, now u need to start by running your VM, press the start button and let it load until the VM is ready to use. (btw i know it is in spanish but you can just traduce it or make an english version urself.)

<img src = "assets/18.png" width = "100%">

---

## IMPORTANT

BEFORE RUNNING THE VM install both Tailscale and Sunshine. You may find some tutorials on there, also this is for LineageOS, not for ArkOS or DarkOs!
